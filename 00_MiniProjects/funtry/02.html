Understanding JavaScript Array and Date Functions

Array Methods

JavaScript Arrays have many predefined methods that help manipulate arrays. Below are a few commonly used methods:

1. push() - Adds one or more elements to the end of an array.

let arr = [1, 2, 3];
arr.push(4); // Adds 4 to the end of the array
console.log(arr); // Output: [1, 2, 3, 4]
Output: [1, 2, 3, 4]

2. pop() - Removes the last element of an array.

let arr = [1, 2, 3];
arr.pop(); // Removes 3 from the array
console.log(arr); // Output: [1, 2]
Output: [1, 2]

3. map() - Creates a new array with the results of calling a provided function on every element in the calling array.

let arr = [1, 2, 3];
let squared = arr.map(x => x * x);
console.log(squared); // Output: [1, 4, 9]
Output: [1, 4, 9]

4. filter() - Creates a new array with all elements that pass the test implemented by the provided function.

let arr = [1, 2, 3, 4, 5];
let evenNumbers = arr.filter(x => x % 2 === 0);
console.log(evenNumbers); // Output: [2, 4]
Output: [2, 4]

5. forEach() - Executes a provided function once for each array element.

let arr = [1, 2, 3];
arr.forEach(x => console.log(x)); 
// Output: 1 2 3 (prints each element in the array)
Output: 1 2 3 (printed individually)

Date Methods

The JavaScript Date object allows you to work with dates and times. Below are some commonly used methods:

1. Date.now() - Returns the current timestamp in milliseconds.

let currentTime = Date.now();
console.log(currentTime); // Output: Current timestamp in milliseconds
Output: 1672287600000 (varies depending on when it's executed)

2. getFullYear() - Returns the year (4 digits) of the specified date.

let date = new Date();
let year = date.getFullYear();
console.log(year); // Output: Current year (e.g., 2024)
Output: 2024 (or the current year)

3. getMonth() - Returns the month (0-11) in the specified date.

let date = new Date();
let month = date.getMonth(); 
console.log(month); // Output: Current month (0 = January, 1 = February, etc.)
Output: 11 (for December)

4. getDate() - Returns the day of the month (1-31) of the specified date.

let date = new Date();
let dayOfMonth = date.getDate();
console.log(dayOfMonth); // Output: Current day of the month (e.g., 26)
Output: 26 (or the current day of the month)

5. getHours() - Returns the hour (0-23) of the specified date.

let date = new Date();
let hour = date.getHours();
console.log(hour); // Output: Current hour (0-23)
Output: 15 (or the current hour)
