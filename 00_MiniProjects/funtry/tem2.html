Array & Date Methods in JavaScript

Array Methods

push() - Add to end:

let arr = [1, 2]; arr.push(3); console.log(arr); // [1, 2, 3]
[1, 2, 3]

pop() - Remove last:

arr.pop(); console.log(arr); // [1, 2]
[1, 2]

shift() - Remove first:

arr.shift(); console.log(arr); // [2]
[2]

unshift() - Add to start:

arr.unshift(1); console.log(arr); // [1, 2]
[1, 2]

Date Methods

getDate() - Day of month:

let date = new Date(); console.log(date.getDate());
e.g., 26

getDay() - Day of week:

console.log(date.getDay());
e.g., 5 (Friday)

getFullYear() - Full year:

console.log(date.getFullYear());
e.g., 2024

toLocaleDateString() - Local date string:

console.log(date.toLocaleDateString());
e.g., "12/26/2024"
